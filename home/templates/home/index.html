Caro (a) usuário (a), seja bem-vindo (a) ao Banco de dados Ambientais do Instituto de Estudos Climáticos (IEC).

O Instituto de Estudos de Mudanças Climáticas do ES (IEC-ES) consolidou bases de dados históricas de clima para o ES e projeções para os cenários futuros de mudanças climáticas. A Plataforma de Dados Ambientais (PDA) tem o objetivo de permitir que pesquisadores, gestores públicos e privados tenham acesso a informações de históricas do clima do ES e projeções de mudanças climáticas futuras com base nos mais recentes estudos científicos disponíveis, incluindo dados de precipitação, umidade, temperatura e demais dados climáticos para orientar as pesquisa na área e os processos de gestão e tomada de decisão quanto à ações de preparação, prevenção e/ou de mitigação dos efeitos das mudanças climáticas previstas.

Nesta plataforma os pesquisadores e a sociedade civil poderão encontrar, gratuitamente, uma compilação das informações de diversas outras plataformas do Brasil e no Mundo e estão disponibilizadas da seguinte forma: Dados Observacionais, Dados de Reanálise, Dados Simulados, Dados de Satélite, Dados de uso e cobertura da terra entre outros, em várias escalas temporais. Na PDA/IEC, o usuário poderá acessar as informações e realizar suas consultas por meio do Portal do Mapa interativo, utilizando o navegador que melhor lhe convir. Mais detalhes sobre a Plataforma de Dados Ambientais do IEC podem ser obtidas aqui.

Os dados históricos incluem series históricas desde 1925 até a data atual, obtidas das estações de monitoramento da Instituto Nacional de Meteorologia (INMET), Centro Nacional de Monitoramento e Alerta de Desastres Naturais (CEMADEN), Agência Nacional de Águas (ANA). Os dados de projeções para os cenários futuros de mudanças climáticas são obtidos pelo modelo regional Eta-CPTEC do Instituto Nacional de Pesquisas Espaciais (INPE).

Nesta categoria podem ser obtidos conjuntos de dados diários observados, para o estado do Espírito Santo, a partir da rede de estações convencionais e automáticas de instituições nacionais, tais como: temperatura, precipitação, vazão, evapotranspiração, radiação solar, vento e umidade relativa do ar. Estão disponibilizados nesta base de dados:

Fontes:

  • Dados da Rede de Estações Meteorológicas Automáticas do INMET desde 2000 até a presente data e dados da Rede de Estações Meteorológicas Convencionais do INMET desde 1924 até a presente data.

  • Dados de Pluviômetros e Fluviométricos da Agência Nacional de Águas (ANA) desde 1925 até 2019.

Estes conjuntos de dados podem ser obtidos no formato csv (comma separated values, que podem ser usados, por exemplo, no Excel) e em formato NetCDF (.nc), que podem ser lidos em softwares como Matlab, Grads, GIS ou utilizando linguagem de programação como Python, Fortran, R, entre outras. Os dados disponíveis também podem ser obtidos diretamente das fontes indicadas.

Os dados provenientes de reanálises meteorológicas combinam os dados medidos e fornecidos pelos institutos nacionais de meteorologia com dados obtidos a partir de modelos de circulação global de alta resolução, desta forma reanálises são um conjunto de dados que combinam informações a partir de observações de estações meteorológicas, modelagem, imagens de satélites e utilizam análises estatísticas na reconstrução da base de dados. Nesta categoria podem ser obtidos conjuntos de dados de reanálises de instituições nacionais e internacionais para o Estado do Espírito Santo. Os dados são disponibilizados em ponto de grade ou pixel que cobrem o Estado do Espírito Santo.

  • CHIRPS: dados mensais disponibilizados pela University of California desde 1981 até a presente data, com Resolução Espacial: 0.05°

  • ERA5: Os dados estão disponíveis em grades regulares de latitude-longitude com resolução espacial de 0.25°, com resolução temporal de 1979 até a presente data.

Assim como nos dados observacionais, as informações das reanálises podem ser obtidas no formato csv (comma separated values, que podem ser usados, por exemplo, no Excel) e em formato NetCDF (.nc), que podem ser lidos em softwares como Matlab, Grads, GIS ou utilizando linguagem de programação como Python, Fortran, R, entre outras.

Nesta categoria podem ser obtidos dados mensais de Cenários Climáticos das projeções de mudança do clima para a América do Sul regionalizadas pelo modelo Eta do Centro de Previsão de Tempo e Estudos Climáticos (CPTEC) do INPE, no âmbito do ProjETA (https://projeta.cptec.inpe.br). As projeções são baseadas no modelo regional Eta-CPTEC com resolução espacial de 5km aninhado ao modelo global HadGen2-ES para o Estado do Espírito Santo. Esses dados compreendem simulações para o período que vai desde 1960 até 2099.

Os dados estão divididos em três conjuntos principais: Período histórico (1960-2005), e dois cenários de emissões de gases de efeito estufa (GEE), o cenário RCP 4.5 (médias emissões de GEE) e o cenário RCP 8.5 (altas emissões de GEE) desde 2006 até 2099. Todos os dados podem ser obtidos por pixel (coordenada geográfica) ou integrado para cada município do estado. Estes conjuntos de dados podem ser obtidos no formato csv (comma separated values, que podem ser usados, por exemplo, no Excel) e em formato NetCDF (.nc), que podem ser lidos em softwares como Matlab, Grads, GIS ou utilizando linguagem de programação como Python, Fortran, R, entre outras.
